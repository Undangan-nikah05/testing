# Panduan Singkat — Undangan Pernikahan

File yang dibuat:
- undangan.html — template undangan digital (bisa langsung dibuka di browser)
- undangan.txt — teks undangan (formal & santai) dan catatan singkat

Cara kustomisasi cepat:
1. Buka [undangan.html](undangan.html) di editor teks.
2. Di bagian akhir file ada objek `data` berisi `bride`, `groom`, `dateLabel`, `time`, `venue`, `mapUrl`, `formal`, `casual`, `rsvp`.
3. Ganti nilai-nilai tersebut sesuai detail resepsi Anda.
4. Simpan lalu buka file di browser untuk melihat hasil.

Cetak atau buat PDF:
1. Buka file di browser.
2. Tekan `Ctrl+P` (Windows/Linux) atau `Cmd+P` (macOS).
3. Pilih "Save as PDF" atau printer jika ingin mencetak fisik.

Ingin saya isi detail acara Anda sekarang? Beri tahu saya:
- Nama mempelai (wanita & pria)
- Hari, tanggal, dan jam
- Nama dan alamat tempat
- Kontak RSVP
- (Opsional) Link Google Maps dan foto
